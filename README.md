# XyraNet Reseller Autoseller

Autoseller for XyraNet Wholesale API with Plati.Market/Digiseller and GGsel integration.

## Features

- Plati.Market/Digiseller and GGsel sales processing.
- Digiseller unique-code chat flow.
- Product mapping by marketplace, lot, button/variant, action, tariff, and parameters.
- Actions: new subscription, renewal, reissue, LTE traffic, IP limit.
- Web panel for mappings, settings, templates, variables, statistics, sales, and bot access.
- Telegram admin bot with the same operational controls, statistics, sales, settings, and notifications.
- Admin notifications for new purchases, chat messages, pending actions, successful delivery, and errors.
- Durable SQLite storage with idempotent deliveries, pending operations, and order event history.
- Production deployment with localhost-only app binding, HTTPS reverse proxy, or SSH-tunnel access without public IP.

## Quick Ubuntu Install

After publishing the repository to GitHub:

```bash
curl -fsSL https://raw.githubusercontent.com/OWNER/REPO/main/scripts/install-linux.sh | sudo env REPO_URL=https://github.com/OWNER/REPO.git bash
```

From a cloned repository:

```bash
sudo bash scripts/install-linux.sh
```

The installer asks for required credentials and can configure either:

- domain + HTTPS through nginx and Let's Encrypt;
- no domain/no public IP with local-only panel access through SSH tunneling.

Full deployment instructions: [DEPLOY.md](DEPLOY.md).

## Windows Local Run

PowerShell:

```powershell
py -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
Copy-Item .env.example .env
notepad .env
python run.py
```

Open:

```text
http://127.0.0.1:8095
```

```env
ADMIN_TOKEN=change-me
MARKETPLACE_WEBHOOK_SECRET=change-me-too
DATABASE_PATH=data/reseller.sqlite3
ENABLE_TELEGRAM=true
```

Marketplace and XyraNet API keys can be filled in `.env` or through the web panel.

## Webhooks

Use only HTTPS webhooks on production:

```text
https://your-domain.example/webhooks/plati?secret=YOUR_WEBHOOK_SECRET
https://your-domain.example/webhooks/ggsel?secret=YOUR_WEBHOOK_SECRET
https://your-domain.example/webhooks/plati/text?secret=YOUR_WEBHOOK_SECRET
```

`MARKETPLACE_WEBHOOK_SECRET` is generated by you or by the installer. It protects webhook endpoints from fake external requests.

## No Domain / No Public IP

Keep the app private and connect through SSH:

```bash
ssh -L 8095:127.0.0.1:8095 user@server
```

Then open locally:

```text
http://127.0.0.1:8095
```

The browser-to-localhost part is HTTP, but the server connection is encrypted through SSH and the panel is not exposed to the Internet.

## Telegram Failure Safety

Telegram polling is supervised separately from the web app. If Telegram is blocked or unavailable, the web panel and marketplace processing continue working; the bot retries with backoff.
